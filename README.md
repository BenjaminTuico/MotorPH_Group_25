# MotorPH Payroll & Employee Management System

A Java Swing-based desktop application designed for MotorPH to streamline employee data management, secure user authentication, and automate payroll/salary computations.

- Features

* **Secure Login Window**: Multi-role authentication that routes users to either the Admin or Employee view depending on credentials.
* **Admin Dashboard**: Centralized panel for HR/Admins to search for staff records, view core company details, and track real-time system metrics.
* **Employee Dashboard**: A private portal where individual employees can securely view their personal profile, active status, and position details.
* **Salary Computation Module**: Automated payroll processing that handles individual employee lookups as well as batch calculations for the entire company roster, complete with neat tabular reporting.
* **Manage Employee Records**: Full data management tool allowing administrators to update credentials, modify salary tracks, remove entries, or instantly reverse deletions via an Undo framework.
* **CSV Database Backend**: Powered by structured flat-file spreadsheets (`EmployeeData.csv`, `Attendance.csv`) to parse, store, and sanitize record rows effortlessly.
